### 快捷键
```
Option-Enter 运行本单元,在其下插入新单元 新单元默认为编辑模式
# 下面两个其实一样
Control-Enter 运行本单元
Shift-Enter 运行本单元,选中下个单元 新单元默认为命令模式

command + MY (MY一块按)

command + MB 在下方插入
command + MA 在上方插入
```
