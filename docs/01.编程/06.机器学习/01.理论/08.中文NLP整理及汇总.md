## spacy

```
# https://blog.csdn.net/weixin_42486623/article/details/124976040
spacy是Python自然语言处理（NLP）软件包，可以对自然语言文本做词性分析、命名实体识别、依赖关系刻画，以及词嵌入向量的计算和可视化等

# 下载
python -m spacy download zh_core_web_sm

tok2vec: 分词
tagger: 词性标注
parser: 依存分析
senter: 分句
ner: 命名实体识别
attribute_ruler: 更改属性映射(没有具体了解)
```
